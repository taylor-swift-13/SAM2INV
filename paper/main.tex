\begin{abstract}
Loop invariant generation is a central challenge in program verification.
Existing LLM-based approaches rely on extensive SMT solving, complex workflows, and the most powerful closed-source models, yet still struggle with this task---which demands logical reasoning, program comprehension, and mastery of formal annotation syntax.
We present \sam{}, a system that trains a small, locally deployable model to generate high-quality numerical loop invariants through offline knowledge distillation with verifier feedback.
To overcome training-data scarcity, we design \lf{}, a probabilistic DSL constrained by semantic templates that synthesises structured numeric loop programs at arbitrary scale.
We then build a multi-stage sampling-and-filtering pipeline: $N$ parallel LLM calls are decomposed into conjuncts, unioned into a single pool, and filtered through syntax validation, execution-trace semantic checking, semantic deduplication, Houdini formal pruning via Frama-C/WP, and trivial-invariant exclusion---producing certified annotations.
Rather than applying reinforcement learning---whose binary verifier reward is too sparse, reward-hackable, and ineffective for weak base models---we adopt a distillation strategy: a lightweight teacher model explores multiple candidates per program during data preparation; the verification pipeline filters them into chosen and rejected examples with reverse chain-of-thought synthesis; and the student is fine-tuned via SFT followed by DPO, with on-policy negative-sample augmentation, transferring the teacher's exploration benefit with no human annotation or online reward model.
\end{abstract}
