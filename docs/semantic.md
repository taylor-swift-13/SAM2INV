

## 1) 计数器推进 + 仿射累加（affine accumulator）

**代码骨架**

```c
i = 0; x = x0;
while (i < N) { i++; x += c; }         // 或 x += a*i + b（但常见是常数）
```

**典型不变量**

* `x = x0 + c*i`
* 等价线性式：`x - c*i = x0`
* 辅助：`0 <= i <= N`

---

## 2) 和/差守恒（conservation: x+y 或 x−y 不变）

**代码骨架**

```c
while (cond) { x += a; y -= a; }   // 和守恒
// 或 { x += a; y += a; }          // 差守恒
```

**典型不变量**

* `x + y = const`（补偿式更新）
* `x - y = const`
* 常配合界：`x>=0`、`y>=0` 或 `i` 的范围

---

## 3) “扣账式”预算变量（budget / countdown）

**代码骨架**

```c
t = T;
while (t > 0) { t -= k; x += c; }
```

**典型不变量**

* `k*x + c*t = const`（或 `x = x0 + c*(T - t)/k` 的线性等价式）
* 单调性：`t` 非增且 `t>=0`，`x` 非减

---

## 4) 线性递推（x 依赖 y 的加法递推，但仍保持线性不变量）

**代码骨架**

```c
while (i < N) { x = x + y; i++; }     // y 不变
// 或 y 也按常数更新： y += d
```

**典型不变量**

* 若 `y` 常量：`x = x0 + y*i`
* 若 `y` 仿射：`y = y0 + d*i`，则 `x` 会变成二次（通常不在 linear）；因此 linear 里常把它改写成**额外状态变量**让整体仍线性（比如维护差值/剩余量）

---

## 5) 分支里的线性守恒（piecewise affine but same invariant）

**代码骨架**

```c
if (p) { x += a; y += b; }
else   { x += a'; y += b'; }
```

但两个分支都保持同一个线性组合不变。

**典型不变量**

* `αx + βy = const`（选 α,β 使得两条更新都抵消）
* 再加上路径条件带来的范围约束（如 `x>=0`、`i<=N`）

---

## 6) 线性“商余模拟/计数展开”（用加减模拟整除/覆盖）

有些 linear 集会出现“用循环加减模拟 `x = q*y + r`”的版本（只用加减和比较，避免乘法/除法）。

**典型不变量**

* `x = q*y + r`（或 `x - q*y - r = 0` 的线性化形式里，`q*y` 若禁止乘法就会用等价状态展开；常见做法是用计数器把 `q*y` 展开成重复加法）
* `0 <= r < y`、`q` 非减

---

# 如果你要把这些范式用于“采样/蒸馏不变量”

对 `linear` 来说最划算的候选空间通常就是两类：

* **等式模板**：`Σ a_j * v_j = c`
* **不等式模板**：`Σ a_j * v_j ≤ c` + 变量区间 `L≤v≤U`

并把系数限制在小整数集合（例如 `{-2,-1,0,1,2}`），变量只取 loop-head 出现的 3~6 个关键变量（`i,x,y,t,q,r` 等），命中率会非常高。


从 `NLA_lipus` 这一组 C 程序（每个文件都带了 post-condition 断言）里，数值循环大概反复出现下面这些**“范式（loop schema）”**——它们对应的循环不变量形状也高度相似。

---

## 1) 多项式递推 / 有限差分：用线性更新生成高次多项式

典型结构：维护 `x,y,z,...` 若干“差分项”，每轮做 `x+=y; y+=z; z+=const` 这类更新，最后 `x` 变成 `n^3`、`y` 变成 `O(n^2)` 等闭式多项式。
例：`x=x+y; y=y+z; z=z+6`，最终断言 `x==n^3`，`y==3n^2+3n+1`，`z==6n+6`。 ([GitHub][1])

**常见不变量形状**：

* `x = P(n)`、`y = Q(n)`、`z = R(n)`（P/Q/R 为多项式）
* 或用差分关系：`y = Δx`，`z = Δy`，`Δz = const`

---

## 2) 求和范式：累加器 + 计数器（Σ1、Σi、Σi²、Σi³…）

典型结构：`c` 从 0 走到 `k`，每轮 `y++`，然后 `x += f(y)`；`f` 是 1 / y / y² / y³…
例：

* `x = x + y` ⇒ `2*x = k^2 + k`（三角数） ([GitHub][2])
* `x = x + y*y` ⇒ `6*x = 2k^3 + 3k^2 + k`（平方和） ([GitHub][3])
* `x = x + y*y*y` ⇒ `4*x = k^4 + 2k^3 + k^2`（立方和） ([GitHub][4])

**常见不变量形状**：

* `c` 的范围：`0<=c<=k`
* `y=c` 或 `y=c+const`
* `x = Σ_{i=1..c} f(i)`，并进一步化成闭式多项式

---

## 3) 几何级数 / 线性递推：`x = x*z + a`，`y = y*z`

典型结构：一边乘 `z` 迭代，一边做 affine 更新（相当于展开得到几何级数）。
例：

* `x = x*z + 1; y = y*z`，断言是一个等价于几何级数闭式的多项式恒等式 ([GitHub][5])
* 更一般：`x = x*z + a; y = y*z`，断言 `z*x - x + a - a*z*y == 0`（同样是闭式恒等式） ([GitHub][6])
* 也有 `y` 初始化成 `z` 或 `1` 的变体。 ([GitHub][7])

**常见不变量形状**：

* `y = z^c`（或 `y = z^{c-1}`）
* `x = a * (z^c - 1)/(z-1)`（用等价恒等式表达以避免除法）

---

## 4) “手工除法/取模”：`q,r` 模拟 `x = q*y + r`

典型结构：维护 `q`（商）和 `r`（余数），让 `r` 在 `[0, y-1]` 循环，溢出就 `q++`，直到覆盖到 `x`。
例：`while(x > y*q + r) { if(r==y-1){r=0;q++;} else r++; }`，断言 `q == x / y` 或 `x == q*y + r`。 ([GitHub][8])
变体：引入 `t=A`，每轮 `t--` 同时推进 `q,r`。 ([GitHub][9])

**常见不变量形状**：

* `x = q*y + r + t`（若有剩余计数 `t`）
* `0 <= r < y`
* 单调性：`q` 非减、`t` 非增

---

## 5) 二进制乘法（Russian peasant / shift-add）

典型结构：按 `y` 的二进制位迭代：若奇数就加到累加器，然后 `x*=2; y/=2`。
例：`if(y%2==1){z+=x; y--; } x=2*x; y=y/2;`，断言 `z==a*b`。 ([GitHub][10])

**常见不变量形状**：

* `z + x*y == a*b`（最经典）
* 以及 `y>=0`、`x` 逐步放大、`y` 逐步缩小确保终止

---

## 6) 扩展欧几里得/Bezout 风格：差减版 gcd + 线性系数跟踪

典型结构：`a,b` 做差减版 gcd（反复 `a-=b` 或 `b-=a`），同时维护系数矩阵/向量 `(p,q,r,s)` 保持线性关系与行列式不变。
例：循环里更新 `a,b` 和 `p,q,r,s`，断言包含：

* `p*s - r*q == 1`（行列式不变） ([GitHub][11])
* `a == y*r + x*p`、`b == x*q + y*s`（线性组合保持） ([GitHub][12])
* 以及由这些推出的交叉乘积恒等式。 ([GitHub][13])

**常见不变量形状**：

* `a = p*x + r*y`，`b = q*x + s*y`
* `det([[p,q],[r,s]]) = 1` 或 `-1`
* `gcd(a,b) = gcd(x,y)`（隐含）

---

## 7) 二进制分解 + 权重累加（更“代数化”的乘法）

比 5) 更复杂的“按位/按 parity 分支”的乘法或计数：反复对 `a,b` 做 `/2`、`-1`，同时维护权重 `p`（例如乘 4）与累加器 `q`。
例：根据 `a,b` 奇偶四种情况更新，最后断言 `q == x*y`。 ([GitHub][14])

**常见不变量形状**：

* `q + p * (a*b 的某种剩余表达) == x*y`
* `p` 是 `2^t` 或 `4^t` 这类权重

## 1) 线性计数器 + 线性累加（把加法堆成乘法）

**形态**：`w++` 作为计数器，`z += x` 作为累加器
**典型不变量**：`z == x*w`（或最终 `z == x*y`）
样例：`main27`（`z += x; w += 1;`）([GitHub][1])、`main31/32`（`sn == i*a` 这类）([GitHub][2])

---

## 2) 二重累加/前缀和（线性 → 三角数/多项式）

**形态**：`z += x` 同时 `p += z`（把等差数列的和再求和）
**典型不变量/后置条件**：`p == x * y*(y+1)/2` 一类闭式
样例：`main28`([GitHub][3])

---

## 3) “幂和”/多项式闭式（Σ i、Σ i²、Σ i³、Σ i⁴、Σ i⁵…）

**形态**：循环里 `y++`，然后 `x += y^d`（d=1..5 常见）
**典型不变量**：`x` 满足某个关于 `k` 的多项式恒等式（把求和写成闭式）
样例：

* `Σ i`：`main15`([GitHub][4])
* `Σ i^2`：`main16`([GitHub][5])
* `Σ i^3..i^5`：`main17~19`([GitHub][6])

---

## 4) 仿射递推 / 几何级数（x = x*z + a, y = y*z）

**形态**：一边做“乘 z”，一边做“乘 z 再加常数”，常见于几何级数/线性递推
**典型不变量**：形如 `x*z - x - y + 1 == 0` 或 `z*x - x + ... == 0` 的代数关系
样例：`main9~11`([GitHub][7])

---

## 5) 商/余数循环（除法算法：q,r 的逐步归纳）

**形态**：维护 `q`（商）和 `r`（余数），让 `r` 在 `[0, y-1]` 循环进位
**典型不变量**：`x == q*y + r`，并推到 `q == x / y`
样例：`main2~4`([GitHub][8])、`main12`（用 `t` 把 A 消耗掉的版本）([GitHub][9])

---

## 6) Euclid / 扩展欧几里得风格（差值迭代 + 线性组合保持不变）

**形态**：`a,b` 通过 `a-=b` / `b-=a` 做 gcd 风格收敛，同时维护系数矩阵 `(p,q;r,s)`
**典型不变量**：

* 行列式常量：`p*s - r*q == 1`
* 线性组合表示：`a == y*r + x*p`、`b == x*q + y*s` 等
  样例：`main5~8`([GitHub][10])

---

## 7) 二进制乘法 / “俄罗斯农民乘法”（按位拆分 + 翻倍/减半）

**形态**：`if (y%2==1) z += x; x=2*x; y=y/2;`
**典型不变量**：`z + x*y == a*b`（或等价变形），最后推出 `z==a*b`
样例：`main14`([GitHub][11])

---

## 8) “逐次减去奇数” 的平方根/界限循环（odd-sum ↔ square）

**形态**：用 `v += 2` 生成奇数序列，然后反复 `r -= v` 或 `s += (2a+1)` 来逼近平方
**典型不变量/后置条件**：`(r-1)^2 <= a`、`n < (a+1)^2` 这种“平方界”
样例：`main20`([GitHub][12])、`main36`([GitHub][13])、`main33`（更代数化的版本）([GitHub][14])

---

## 9) 乘积/阶乘式累积（w *= x；或区间连乘）

**形态**：`w = w * x`（可能在区间上从 1 乘到 y），常用引入“差一个因子”的辅助变量
**典型不变量**：`w == z * y` 或 “w 是连乘，z 是缺末项的连乘”
样例：`main26`([GitHub][15])、`main25`（未知迭代次数但保持 `z==w*x`）([GitHub][16])

---

## 10) 不等式保持型（比如 Cauchy-Schwarz / 单调性）

**形态**：每轮都加同一种结构项（如 `x^2, y^2, x*y`），最后得到全程保持的不等式
**典型不变量/后置条件**：`z*w >= p*p` 这类二次不等式
样例：`main29/30`([GitHub][17])
（另外像 `y <= x*x` 也是典型“单调+界”的不等式范式）([GitHub][18])

**典型不变量**

* `x = q*y + r`（或 `x - q*y - r = 0` 的线性化形式里，`q*y` 若禁止乘法就会用等价状态展开；常见做法是用计数器把 `q*y` 展开成重复加法）
* `0 <= r < y`、`q` 非减

# 如果你要把这些范式用于“采样/蒸馏不变量”

对 `linear` 来说最划算的候选空间通常就是两类：

* **等式模板**：`Σ a_j * v_j = c`

## 常见数值循环范式（面向不变量/合成）

### 1) 线性计数器 / 仿射更新（Affine recurrence）

**更新形态**

* `i := i + c` / `i := i - c`
* 多变量：`x := x + a`, `y := y + b`, 或 `x := x + y + c`

**典型不变量**

* 区间单调：`i >= i0`、`i <= bound`
* 线性关系：`x - y = const`、`a*x + b*y = const`
* 与迭代次数关联：`x = x0 + k*a`

> 这是最常见的“骨架”，很多非线性例子也是在它上面叠加乘法/平方项。

---

### 2) 线性组合守恒（Linear invariant / 常量保持）

**更新形态**

* 交换/抵消式：`x := x + 1; y := y - 1`
* 轮换：`t:=x; x:=y; y:=t`（或带偏移）

**典型不变量**

* `x + y = const`
* `x - y = const`
* `x + y + z = const`

---

### 3) 乘法缩放 / 几何递推（Geometric / Multiplicative）

**更新形态**

* `x := x * c`、`x := x / c`（整数除法时常见）
* `x := x * y`（更难）

**典型不变量**

* 若配套计数器 `i := i+1`：`x = x0 * c^i`
* 对数式/指数式关系（工具通常用引入辅助变量或用不等式包住）
* 乘积守恒：`x*y = const`（比如 `x*=2; y/=2` 这类）

---

### 4) 多项式增长：平方/立方项出现（Polynomial invariants）

**更新形态**

* `s := s + i`（累加 → 二次）
* `s := s + i*i`（累加平方 → 三次/四次）
* `x := x + y; y := y + 1`（组合后会出现二次关系）

**典型不变量**

* `2*s = i*(i-1) + const`（三角数）
* `6*s = i*(i-1)*(2*i-1) + const`（平方和）
* 形如 `s = poly(i) + const` 的多项式等式/不等式包络

> 这类是 NLA benchmark 的核心：不变量不是线性的，而是“变量之间满足某个多项式关系”。


### 5) 乘加耦合（Bilinear / Mixed terms）

**更新形态**

* `x := x + y; y := y + z`（会产生交叉项）
* `p := p + x*y` 或 `x := x + y*z`

**典型不变量**

* `x*y - f(other vars) = const`
* `x^2 + y^2`、`x*y`、`x*y + x + y` 这类组合保持或单调

---

### 6) 条件分支更新（Piecewise / If-in-loop）

**更新形态**

* `if (x > 0) x := x - a else x := x + b`
* `if (x%2==0) x:=x/2 else x:=3*x+1`（更极端）

**典型不变量**

* 分段不等式不变量：`x >= 0`、`x <= M`
* “势能函数”单调：`V(x,y,...)` 每轮下降（用来证明终止/界）

> 这类通常需要“不等式 + 分支条件”组合，而不是单一等式。

---

### 7) 最大/最小截断、饱和（Saturating / Clamp）

**更新形态**

* `x := min(x+c, M)` / `x := max(x-c, 0)`
* `if (x > M) x := M`

**典型不变量**

* 边界：`0 <= x <= M`
* 单调性：`x` 非降/非增
* 稳定点：到达边界后保持不变

---

### 8) 欧几里得式收缩（GCD/差分收缩类）

**更新形态**

* `while (x != y) if (x>y) x:=x-y else y:=y-x`
* 或带除法/取模版本

**典型不变量**

* `gcd(x,y) = gcd(x0,y0)`（语义不变量）
* `x>0 ∧ y>0`、`x+y` 单调下降（终止证明）
