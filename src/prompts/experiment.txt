### CRITICAL Rules ###

1. **Boundary Conditions are REQUIRED**:
   - If assert needs `var == target`, you MUST include `var <= target` or `var >= target`
   - Example: assert `n == a+1` with loop `while(n <= a)` needs invariant `n <= a + 1`

2. **Do NOT use loop condition as invariant**:
   - Loop condition becomes FALSE when loop exits
   - WRONG: `loop invariant x > y*q + r;`
   - RIGHT: `loop invariant y*q + r <= x;`

3. **For division algorithms** (assert `x == q*y + r`):
   - MUST include: `loop invariant q*y + r <= x;`
   - MUST include: `loop invariant 0 <= r < y;`

4. **ABSOLUTELY FORBIDDEN SYNTAX**:
